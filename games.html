Game Use Instructions

How do I play a board game?

All of our available board games are located behind the Next House Desk. You will need to check out a game out to be able to use it! Checking out an item is fairly straightforward, and only requires your MIT ID.


First, you will need to approach the desk while a desk worker is present (usually from 8am to 1am). Tell them you'd like to check out a board game, saying specifically which game you'd like to check out. You should present your ID at this point, if the desk worker hasn't already asked for it. The desk worker will then grab the game for you. Most of our board games are held in BIT Box storage, so you might get the board and pieces in separate bags. Once the desk worker hands you the pieces and logs you in the system as having checked out the item, you are all good to go!


Games are checked out for 4 days by default, but you should return the game after you finish playing. There is a limit of 2 checkouts per person. There is no renewal system for games. To return a game, simply hand all the pieces you received back to a desk worker. If you hold onto the board game for more than 4 days, you will receive an email reminder notifying you that you should return the game. Make sure to return the game by this point! And as a general reminder, please take care of the board game while you have it checked out.


How do I play a video game?

All of our available video games are downloaded onto our very own Next Gamers PS4! The PS4 is located in the TFL TV Area of Next House. You are free to use the PS4 as long as no one has the space reserved, which you can check the reservation status of on NextRes. If the space is open, you'll need to check out a controller from the Next House Desk. We have 4 controllers available, so you and up to 3 friends can game on the PS4. The procedure to check out a controller is very similar to checking out a board game (refer to above!).


Once you have acquired one or more PS4 controllers, you can head back to the TFL TV Area and get the PS4 going. To turn on the TV and make sure it has the correct output, refer to the guide found here. Once the TV is on, you can make sure the PS4 is in order by following this PS4 specific guide. After following this, you should be all set to use the PS4!


The PS4 controllers are checked for 4 days by default, but you should return them right after using them, since they are expensive pieces of equipment. To return the controllers, simply bring them to a desk worker and tell them you'd like to return the controller. If you hold onto the controller for more than 4 days, you will receive an email reminder notifying you that you should return it. Make sure to return the controller by this point! And as a general reminder, please take care of the controller and PS4 while using it.
